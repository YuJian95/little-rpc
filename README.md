# little-rpc

一个简单的 `RPC` 框架实现，用于学习 `RPC `原理（远程过程调用）。

## 模块说明

- little-rpc-codec：序列化模块
- little-rpc-common：通用模块
- little-rpc-transport：网络传输模块
- little-rpc-proto：协议模块
- little-rpc-server: 服务端
- little-rpc-client：客户端
- little-rpc-demo：测试样例
